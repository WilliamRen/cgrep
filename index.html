Welcome to CGrep Home-Page.

Cgrep is a grep tool suitable for searching in large code repositories. It supports 30 programming languages and searches that go beyond the mere pattern matching. It enables context-aware filtering for all the languages and semantic searches for C/C++.

Features

  • Multi-threaded with configurable number of jobs.

  • Blazing fast: Running on a single core it is nearly as fast as GNU grep, up to 6x times faster than ack-grep.

  • Multi-line pattern searches.

  • Colors highlighting tokens and strings matched.

  • Recursive searches: Support of language filters and pruned directories.

  • Code Tokenizer: search for simple, prefix, infix, regex and similar string occurrences through the Levenshtein distance (edit distance).

  • Context-aware filters for Awk, C, Cpp, Chapel, Csharp, Css, CMake, D, Erlang, Fsharp, Go, Haskell, Html, Java, Javascript, Latex, Lua, Make, OCaml, ObjectiveC, Perl, PHP , Python, Ruby, Scala, Tcl, Text, Shell, Verilog, VHDL, Vim.

  • Semantic searches enabled for C/C++ language.

  • Customizable output with format string, xml and JSON back-ends.

Libraries

  • In addition to the Haskell platform the following libraries are required: stringsearch, cmdargs and ansi-terminal

Releases
