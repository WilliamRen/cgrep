Welcome to CGrep HomePage.

Cgrep is a grep tool suitable for searching in large code repositories. It supports 27 programming languages and searches that go beyond the mere pattern matching. It enables context-aware filtering for all the languages and semantic searches for C/C++.

Features

  • Multi-threaded: Configurable number of jobs.

  • Blazing fast: Running on a single core it is nearly as fast as GNU grep, up to 6x times faster than ack-grep.

  • Multi-line, multi-pattern searches.

  • Colors hilight for tokens and strings matched.

  • Recursive searches: Support of languages filters and pruned directories.

  • Code Tokenizer: Simple, regex and full word-matching searches through a generic code tokenizer.

  • Context-aware filters: support for the languages Awk, C, Cpp, Csharp, Css, CMake, D, Erlang, Fsharp, Go, Haskell, Html, Java, Javascript, Latex, Make, OCaml, ObjectiveC , Perl, PHP , Python, Ruby, Scala, Tcl, Shell, Verilog, Vim.

  • Semantic searches enabled for C/C++ languages.

Libraries

  • In addition to the Haskell platform the following libraries are required: stringsearch, cmdargs and ansi-terminal
