Cgrep

Cgrep: a context-aware grep for source codes

View project onGitHub

Welcome to CGrep HomePage.

Cgrep is a grep tool suitable for large code repositories. It supports multi-threads, 27 programming languages, and searchers that go beyond the mere pattern matching. It enables context-aware searches for all languages and semantic searches for C/C++.

Features

  • Multi-line, multi-pattern and multi-thread searches.

  • Colors hilight for tokens and strings matching.

  • Recursive searches with prune directories and language filters.

  • Simple, regex and full word-matching searches through a generic code tokenizer.

  • Support for context-aware filters for the following languages: Awk, C, Cpp, Csharp, Css, CMake, D, Erlang, Fsharp, Go, Haskell, Html, Java, Javascript, Latex, Make, OCaml, ObjectiveC , Perl, PHP , Python, Ruby, Scala, Tcl, Shell, Verilog, Vim.

  • Semantic searches for C/C++ languages.

Libraries

  • In addition to the Haskell platform, the following libraries are required:
    • stringsearch, cmdargs and ansi-terminal
